# Hyperbolic residual finiteness

[← Back to repository README](../../README.md)

The current exploratory problem in geometric group theory is:

> Is every finitely generated word-hyperbolic group residually finite?

Equivalently, can one construct a word-hyperbolic group `G` with a nontrivial element
`g` in its finite residual

```math
R_f(G) = \bigcap_{N \triangleleft G,\ [G:N]<\infty} N?
```

Such an element is killed by every homomorphism from `G` to a finite group, so it would give a
counterexample. The prompt sequence is in
[prompts/hyperbolic_residual_finiteness](../../prompts/hyperbolic_residual_finiteness/), and the
resulting research scenarios are recorded under [plan](../../plan/).
