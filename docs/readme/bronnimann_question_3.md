# Brönnimann Question 3

[← Back to repository README](../../README.md)

The `Groups` library is the home for formalization work around geodesics in finitely generated
groups. The current Question 3 front is centered on the fully internal witness-packaging theorem
and the supporting definitions needed to formulate Bodart's virtually Engel example precisely.

The main modules are:

- `Groups.GeodesicGrowth` for generic definitions of solvable word problem and
  rational / irrational geodesic growth;
- `Groups.VirtuallyEngelGroup` for Bodart's virtually Engel presentation and imported witness
  facts;
- `Groups.BronnimannQuestion3` for the fully local witness-packaging theorem.

The older `Groups.HeisenbergGroup` development remains in the repository as adjacent background,
but it is no longer the main framing for this front. A standalone mathematical note on the
question and its resolution (independent of the Lean development) is available as
[PDF](../bronnimann_question_3/proof.pdf) / [source](../bronnimann_question_3/proof.tex).

### Formalization structure

The Lean development mirrors the mathematical note but adds the machine-checked internals. The
witness is Bodart's virtually Engel group

```
G = ⟨ a, t | t² = 1,  C = [aᵗ,[a,aᵗ]],  [C,a] = 1,  [C,aᵗ] = 1 ⟩,
    aᵗ = t⁻¹at,  C = [a,[a,aᵗ]],   alphabet  S = {a, a⁻¹, t}.
```

**Computable coordinate model.** The nilpotent part is modelled by integer coordinates
`(u, v, A, B) ∈ ℤ⁴` (two diagonal directions `u,v`, an area term `A`, and three times a
barycentric term `B`) with multiplication

```
(u,v,A,B)(u',v',A',B') =
  (u+u', v+v', A+A'+vu'−uv', B+B'+3(u−v)A'+(2(u−v)+(u'−v'))(vu'−uv')),
```

and the order-two reflection `σ(u,v,A,B) = (−v,−u,−A,−B)`. The group model is the semidirect
product `CoordGroup = EngelCoords ⋊ Symmetry`; `a ↦ (1,0,0,0)`, `t ↦` the symmetry. The four
relators of the presentation are verified in this model, giving a canonical homomorphism
`toCoordGroup`. The lattice generated by the diagonal steps is cut out internally by the two
congruences `A + uv ≡ 0 (mod 2)` and `B − 3A(u−v) − u²v + 2uv² ≡ 0 (mod 6)`, proved stable under
multiplication, inversion, and reflection. Normal forms use the Hall-convention commutator
`a⁻¹(aᵗ)⁻¹a aᵗ` (not mathlib's literal `[a,aᵗ]`), so that each lattice point is reconstructed
exactly from an explicit Hall normal-form word.

**Axiom status of `positive_answer`** (`A1`/`A2`/`A3` refer to the three original witness facts):

- **A1 — injectivity of `toCoordGroup`** (`toCoordGroup_injective`): **discharged as a theorem.**
  Injectivity is reduced to a set-theoretic retract (Hall normal-form reconstruction) being a
  left inverse of `toCoordGroup`, proved by a free-group word induction using the Hall
  commutation identities and the `T`-conjugation identity. Solvable word problem follows, since
  `CoordGroup` has decidable equality.
- **A3 — asymptotic-to-irrationality bridge** (`irrationalGeodesicGrowth`): **discharged as a
  theorem** in `Groups.LinearRecurrenceGrowth`. A sub-exponential integer sequence satisfying a
  constant-coefficient linear recurrence is polynomially bounded (companion-operator /
  generalized-eigenspace analysis; eigenvalues of modulus `≤ 1`), so an intermediate-growth
  sequence is irrational.
- **A2 — Bodart's growth estimate** (`theorem_4`): **proved theorem**, stated as the growth
  equivalence `γ(n) ≍ exp(n^(3/5) log n)` via
  `LinearRecurrenceGrowth.CoarseGrowth.CoarseEquiv`. It is assembled from two named sub-axioms
  isolating the deep metric-geometric inputs:
  - `theorem_4_upper`: `γ(n) ≤ c₂·exp(n^(3/5) log n)` eventually (Stoll [Sto98] / Pansu [Pan83]
    via Bodart §2–§3);
  - `theorem_4_lower`: `c₁·exp(κ·n^(3/5) log n) ≤ γ(n)` eventually, some `κ ∈ (0,1]` (Bodart §4
    explicit geodesic family).

  The lower half only controls the exponent up to the smaller constant `κ < 1`; a *strict*
  two-sided bound against a single `exp(n^(3/5) log n)` would be **false** (the ratio
  `exp(κ·n^(3/5)logn)/exp(n^(3/5)logn) → 0`). The argument-rescaling freedom of `≍` reconciles
  the two constants (pick `k` with `κ·k^(3/5) ≥ 1`), so `theorem_4` follows. `theorem_4_upper`
  and `theorem_4_lower` are the sole remaining internal-facing axioms; discharging them (from
  scratch, via `Groups.StollPansu` and the polynomial reachability box in
  `Groups.EngelGeodesicGrowth`) is the remaining internal work.
