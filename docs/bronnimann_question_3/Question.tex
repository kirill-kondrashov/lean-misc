\section{Br\"onnimann's Question 3}
\label{sec:Question}

Br\"onnimann asks:
\begin{quote}
Is there a group with solvable word problem and irrational geodesic growth?
\end{quote}

The question is formalized as an existence statement of the following form.

\begin{definition}[Positive answer to Question 3]
One says that Br\"onnimann's Question~3 has a positive answer if there exist:
\begin{itemize}
  \item a finitely generated group given by a presentation $\langle \alpha \mid R \rangle$;
  \item a finite alphabet $\beta$ together with an interpretation of each letter of $\beta$ as an
    element of the presented group;
\end{itemize}
such that:
\begin{enumerate}[label=(\roman*)]
  \item the presented group has solvable word problem, and
  \item the geodesic growth with respect to the chosen alphabet is irrational.
\end{enumerate}
\end{definition}

This formulation records only the existence content of Br\"onnimann's question. No specific
presentation or generating alphabet is built into the statement.
